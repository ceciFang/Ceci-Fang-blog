## 使用 prettiner 格式化代码

使用命令 `npm i prettier -D`

```sh
npm i prettier -D
```

在 package.json 文件里面会出现以下代码表示安装成功:

```JSON
 "devDependencies": {
    "prettier": "^2.5.1"
  }
```

```sh
./node_modules/.bin/prettier --write "src/**/*.js"
```
