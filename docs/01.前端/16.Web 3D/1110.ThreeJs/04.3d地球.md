## npm 安装

```bash
npm install three
```

## 安装轨道控件插件

```bash
npm install three-orbit-controls
```

## 接下来安装加载.obj 和.mtl 文件的插件

```bash
npm i --save three-obj-mtl-loader
```

## 安装渲染器插件

```bash
npm i --save three-css2drender
```
